# SEO Analyzer API

A Node.js + Express API that analyzes basic SEO elements of any website URL.

**Live Demo**: https://seo-project-nhur.onrender.com  
**GitHub**: https://github.com/reemu01/seo-project

---

## Tech Stack
- **Backend**: Node.js, Express.js
- **Web Scraping**: Axios, Cheerio  
- **Deployment**: Render
- **Version Control**: Git, GitHub

---

## Setup Locally
```bash
git clone https://github.com/reemu01/seo-project.git
cd seo-project
npm install
node server.js
